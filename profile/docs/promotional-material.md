## 🎯 研修のコンセプト

**「AI駆動開発全盛の時代でも、基礎力と正しい開発フローを身につけたエンジニアを育成する」**

GitHub Copilot、Cursor、ChatGPTなどのAIツールが普及する中、以下の3つの能力が重要になっています：

1. **AIの間違いを見抜く力** - 基礎知識がないと、AIの出力が正しいか判断できない
2. **開発の正しいフローを知る** - AIに振り回されず、目的を持ってAIを活用できる
3. **しっかりとした土台を作る** - 一度は自分の手を動かして作る経験

**特に、AI駆動開発全盛の時代だからこそ、セキュリティの重要性が増しています。** AIが生成するコードにはセキュリティリスクが潜んでいる可能性があり、それを適切に評価・修正できる能力が不可欠です。本研修では、セキュリティ基礎を共通基礎カリキュラムに組み込み、安全なアプリケーションを開発できるエンジニアを育成します。

## 📚 カリキュラム構成

| カテゴリ | 内容 | 日数 |

| 04. データベース基礎 | SQL、設計、トランザクション | 2-3日 |

| 06. セキュリティ基礎 | OWASP Top 10、認証・認可、AI生成コードのセキュリティ評価 | 1-2日 |
